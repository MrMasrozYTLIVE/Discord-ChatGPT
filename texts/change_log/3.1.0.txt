## Версия: 3.1.0 Release
## Статус: НЕ АКТУАЛЬНА
## Даты релиза:
- 07.09.24 Beta Build 1
- 08.09.24 Beta Build 2
- 08.09.24 Release

### Основные изменения:
- Все ИИ модели теперь автоматически определяются библиотекой g4f.
- В разы больше стабильных провайдеров для основных моделей ИИ: `GPT 3.5, 4, 4о, 4 Mini, LlaMa 3.1, Gemini.
- Команда `/info` заменена на `/about` для лучшего понимания.
- Полностью изменен алгоритм `/draw` теперь запросы отправляются через `g4f.generate` и не требует API.

### Добавлено:
- ИИ модель: `Gemma 2B` от Google.
- ИИ модели для генерации картинок:
  ```
  - SDXL
  - SD v3
  - Playground v2.5
  - FLUX: Core, Anime, Realism, 3D, Disney
  - DALL-E и DALL-E Mini
  - EMI
  ```

### Обновления:
- Системные библиотеки **g4f** обновлены до 3.0.2.6.

### Удалено:
- Ряд неактуальных моделей ИИ
